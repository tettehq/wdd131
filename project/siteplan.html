MY SITE PLAN

SITE NAME

The name of the site is Portfolio Builder

SITE PURPOSE

The purpose of the site is to provide a template using basic HTML, CSS and JavaScript for aspiring web developers to build a simple portfolio that showcases their work.

SCENARIOS

Target audience of the site is web developers.

  • How do I create a simple portfolio to showcase my work as a developer?
  • What items do I need to include in my portfolio?
